# Procurement Constraint Intelligence

A Streamlit-based proof of concept that demonstrates supplier document ingestion, parsing, structured procurement constraint extraction, and an active supplier MOQ and deadline tracker.

## Features
- Upload PDF, DOCX, or TXT files
- Extract readable text from supplier documents
- Generate structured procurement constraint outputs
- Track supplier MOQ and order deadlines
- Support planner review workflow

## Tech Stack
- Streamlit
- PyPDF2
- python-docx
- pandas
